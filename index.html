About Me

After years in academia & non-profits, turns out, writing code is my true calling. A non-traditionalist in many ways, I got my web dev education from Dev Bootcamp in SF. Bootcamp was intense, but the opportunity to learn to code, and in a rad and inclusive atmosphere, was the best decision I've ever made in my life. Post Bootcamp, I am honing my skills to become a great developer. I have a lot to learn, but I am excited about what the future holds.

While I love coding, my passion is first and foremost people. When I code, I'm solving problems to benefit people's lives. Whether it's fixing bugs, implementing a better design, or adding brand new features, I code knowing that whatever I contribute will have an impact on living, breathing people.

When I'm not coding, I like to watch horror movies with basset hounds, brunch with friends, hike in the Oakland Hills, and think about how fantastic Idris Elba would be as the next Doctor Who.

What I've built

Sanctuary

Provides immigrant community with tools to protect themselves against I.C.E. raids. Users can map and track I.C.E. sightings, instantly notify up to 5 contacts via text message if they are being targeted, and access their rights. Site fully translated into Spanish. Role: Project-lead, Full-stack Developer. Stack: Rails, HTML/CSS/Javascript, Materialize, i18n, Google Maps API, Twilio API, PostgreSQL

Remix-It

Allows users to easily create, and send songs to friends. Recipients can contribute to the recording, accessing new samples, creating their own remix. Role: Full-stack Developer. Stack: Sinatra, JavaScript, JQuery, HTML, CSS, Ajax.

Pure CSS Koala

Pure CSS Koala image built by following a tutorial written by Michael Mangialardi the creator of Coding Artist. Stack: HTML/CSS.

Hand Up

Web app helping non-profits find resources, connect wtih volunteers, and receive donations. 2-person team. Role: Front-end developer. Stack: Ruby on Rails, JavaScript, JQuery, HTML, CSS, Ajax.
